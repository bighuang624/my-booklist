## 语言相关

### JavaScript

* 《JavaScript设计模式与开发实践》：这本买了，但是因为涵盖了太多的函数式编程、设计模式等内容，之前看不太懂。也不知道以后有没有机会读了。

### Python

* 《Python cookbook》：只要读研，Python 肯定要学。要将基础打扎实。
